# Digital-breakthrough-the-championship-2022-Altai

Перед участниками [чемпионата](https://hacks-ai.ru/championships/758263) стоит задача предсказать будущий потенциальный статус студента на основе данных нескольких тысяч студентов. Такая прогнозная модель может стать хорошим инструментом при планировании учебной работы в вузах, взаимодействии с работодателями или реализации научных проектов.
